Sobre la materia

Diseño Web es una materia que se dicta durante el primer cuatrimestre del 4º año de la carrera de Licenciatura en Sistemas de Informacion, en la Facultad de Ciencias Exactas Quimicas y Naturales de la Universidad Nacional de Misiones

Unidades

Algunas de las unidades que podemos encontrar en la materia son:

  • Servidores Web Locales
  • Ambientes para el desarrollo de Aplicaciones Web
  • HTML
  • CSS
